ARPANET

ARPA created an experimental network called ARPANET (Advanced Research Projects Agency Network). It was an experimental network, where new applications were tested. It was a way for scientists to communicate quickly.

The (NCP - Network Control Protocol) was the first Arpanet server protocol, which did all the connection procedure between two points. Complex use of file and information exchange and use of remote machines.

After a few years the arpanet was split into two parts: MILNET for information exchange and military file exchange. ARPANET ARPANET with the scientific studies part.

ARPANET and NORSAR

NORSAL, a noroegues satellite system has joined forces with arpanet.

Then came the idea that the whole world needed an open architecture network with a single protocol, and its standards. Then the process of interconnecting networks was called Internet
